# BellezzaByNaomi - Beauty Studio Premium

Landing page profesional para el estudio de estética BellezzaByNaomi.

## Características
- **Diseño Premium**: Estética minimalista y lujosa (Gold & Pink).
- **Interactividad**: Pantalla de revelado cinematográfico, partículas de corazones y animaciones de scroll.
- **Reservas Directas**: Integración con WhatsApp Business para agendar citas automáticamente por servicio.
- **Galería de Servicios**: Despliegue de tratamientos con fotos reales de los trabajos.
- **Responsive**: Totalmente adaptado para móviles y tablets.

## Despliegue en Vercel
Para subir este proyecto a Vercel, sigue estos pasos:
1. Instala la CLI de Vercel: `npm i -g vercel` (si no la tienes).
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el comando `vercel`.
4. Sigue las instrucciones en pantalla para configurar el despliegue.

O simplemente arrastra la carpeta a [Vercel Dashboard](https://vercel.com/new).

## Estructura
- `index.html`: Estructura, estilos (CSS) y lógica (JS).
- `assets/icons/cards/`: Iconos de las categorías principales.
- `assets/images/services/`: Fotos reales de los servicios.
